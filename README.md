# PolicyExpert
Policyexpert Home insurance page automation practice

This is a sample project on PolicyExpert's Home insurance page. 

#Tools used
* Java 8
* Gradle 7.1
* Cucumber
* Selenium

#Run the code
## From command line
Navigate to the root of the project
* `./policyexpert` 
* `gradle cucumber`

## From intellij
* Select the gradle `Tasks->other->cucumber`
